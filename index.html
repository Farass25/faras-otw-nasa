Halo, nama saya Muhammad Rizqi Faras Abyadh. Portofolio ini adalah kanvas digital tempat saya menuangkan ide, logika, dan kecintaan saya pada teknologi. Menggunakan paduan desain yang bersih dan kode yang terstruktur, saya percaya bahwa fungsionalitas harus berjalan beriringan dengan estetika visual.

Bagi saya, menulis kode bukanlah sekadar menyusun teks, melainkan seni memecahkan masalah. Gulir ke bawah untuk melihat perjalanan saya merajut antarmuka web (UI/UX) hingga memprogram mikrokontroler perangkat keras.

Creative Coder

Mengenal Lebih
Dekat

MAN 1 Magelang

Filosofi
Elegan

Banyak yang beranggapan bahwa website teknis harus terlihat gelap, rumit, dan penuh dengan angka hijau bergaya film The Matrix. Saya ingin mematahkan stereotip tersebut. Efek transparansi, bayangan lembut, dan animasi transisi yang mulus membutuhkan pemahaman CSS tingkat lanjut. Hal ini menunjukkan bahwa kesederhanaan desain (simplicity) sebenarnya menyembunyikan kompleksitas eksekusi yang brilian di belakang layar.

Modern UI

Seni
Membongkar

Ketertarikan saya pada hardware sudah dimulai sejak kecil. Alih-alih memainkan mobil-mobilan remote control, saya lebih suka membongkarnya untuk melihat dinamo dan kabel di dalamnya. Fase dekonstruksi ini adalah awal dari pemahaman saya tentang bagaimana listrik diubah menjadi gerakan mekanis.

Seiring berjalannya waktu, kebiasaan "merusak" itu berubah menjadi keterampilan merakit. Memahami jalur kelistrikan, fungsi resistor, dan sirkuit terpadu (IC) memberikan saya dasar yang kokoh sebelum akhirnya saya beralih ke ranah pemrograman siber.

Hardware Builder

Otak dari
Segalanya

Arduino Uno dan ESP32 adalah dua "otak" favorit saya di meja kerja. Mikrokontroler ini berukuran sangat kecil, namun kemampuannya nyaris tak terbatas. Menggunakan bahasa C/C++, saya belajar cara menanamkan logika percabangan, perulangan, dan interupsi ke dalam chip silikon.

Melihat sebuah sistem fisik hidup—lampu berkedip sesuai pola, motor servo bergerak mencari sudut presisi, atau layar LCD menampilkan data karakter—hanya karena serangkaian instruksi teks yang saya ketik, selalu memberikan kepuasan yang tak tertandingi.

Pelajari Arduino

Internet of
Things (IoT)

Keahlian hardware saya mencapai level baru ketika dipadukan dengan modul Wi-Fi, yang melahirkan konsep IoT. Visi terbesar saya adalah otomasi rumah tangga (Smart Home). Bayangkan, Anda bisa mengontrol suhu AC, mematikan lampu, atau mengunci pintu rumah secara *real-time* dari mana pun menggunakan *smartphone*.

Dalam proyek ini, saya menggunakan komponen seperti Relay untuk memutus arus listrik 220V secara aman. Pemrograman IoT tidak hanya sekadar membuat hidup lebih nyaman, tetapi secara signifikan mampu memangkas pemborosan energi listrik secara global.

Smart System

Agrikultur
Cerdas

Selain otomatisasi rumah, saya sangat peduli dengan sektor lingkungan. Salah satu eksperimen saya adalah *Smart Garden*, sebuah sistem yang memantau tingkat kelembaban tanah menggunakan sensor *soil moisture*. Ketika tanah mendeteksi kekeringan, pompa air mini akan menyala sendiri tanpa perlu perintah manual.

Sistem loop tertutup (closed-loop) seperti ini menyelamatkan tanaman dari kekeringan sekaligus mencegah penyiraman air yang berlebihan. Ini adalah contoh sempurna bagaimana teknologi mikrokontroler murah dapat memberikan dampak nyata pada pelestarian alam.

Eco Tech

Data dari
Sensor

Dunia kita penuh dengan data fisik yang tak terlihat, mulai dari suhu, intensitas cahaya, kelembaban, hingga gelombang suara. Sensor adalah mata dan telinga bagi sebuah mesin. Membaca data mentah dari sensor DHT11 (Suhu) atau Ultrasonik (Jarak) merupakan hal yang sangat menarik sekaligus menantang.

Tantangannya terletak pada bagaimana kita memfilter noise (gangguan sinyal). Di sinilah kemampuan matematika dan logika algoritma diuji. Saya harus memastikan sistem komputer tidak mengambil tindakan gegabah akibat pembacaan data sensor yang keliru selama sekian milidetik.

Menyelami
Front-End

Sehebat apapun perangkat keras beroperasi, ia tetap butuh antarmuka pengguna (Dashboard) agar mudah dikontrol oleh manusia biasa. Hal inilah yang menjerumuskan saya dengan bahagia ke dunia Web Development (Front-End). Saya mengubah data sensor yang kaku menjadi grafik batang yang interaktif di layar browser.

Dengan menguasai HTML sebagai tulang punggung struktur, CSS sebagai desainer visual, dan JavaScript sebagai mesin logika penggeraknya, saya memiliki kontrol penuh atas piksel demi piksel di layar Anda. Ini adalah keahlian yang melengkapi portofolio Full-Stack IoT saya.

MDN Web Docs

Empati dalam
UI / UX

Bagi saya, UI/UX (User Interface / User Experience) bukanlah sekadar membuat tombol terlihat indah. UX adalah tentang empati, bagaimana kita bisa memikirkan kemudahan orang lain saat mereka menggunakan sistem yang kita buat. Jangan sampai pengguna kesulitan mencari menu atau tombol navigasi.

Tema terang (light mode) yang dipadukan dengan desain *clean* pada website portofolio ini adalah hasil dari analisis UX. Latar belakang terang dengan ruang kosong (white-space) yang luas secara psikologis mengurangi kelelahan mata dan membuat pengunjung lebih fokus membaca konten.

UX Empathy

Seni Manipulasi
CSS 3D

Untuk benar-benar menguji batas kemampuan CSS, saya merancang sebuah proyek eksperimen berupa ID Card (Lanyard) interaktif. Tanpa menggunakan aplikasi grafis berat atau rendering video, saya menggunakan kode murni untuk menyimulasikan fisika ayunan dan prespektif 3D pada objek di layar web.

Ini bukan sekadar pertunjukan visual, tetapi pembuktian bahwa dengan pemahaman mendalam tentang *transform-origin*, variabel matematika kalkulasi, dan efek *box-shadow*, kita bisa merusak batas dua dimensi pada browser standar. (Silakan cek di menu Lanyard!).

Kesenian
Debugging

Dunia programming identik dengan kesalahan teknis. Sering kali, satu tanda koma yang hilang atau kabel yang tertukar menyebabkan seluruh sistem gagal beroperasi. Proses pencarian sumber masalah yang kita kenal dengan istilah "Debugging" ini adalah rutinitas yang melatih mental pantang menyerah.

Saya belajar bahwa *error message* berwarna merah bukanlah musuh, melainkan petunjuk dari komputer. Kemampuan untuk tetap tenang, membaca dokumentasi, dan melacak logika setapak demi setapak adalah kemampuan paling berharga (soft-skill) yang saya peroleh dari proses coding.
