ARSIP KOMPETENSI

Muhammad Rizqi Faras Abyadh - MAN 1 Magelang

Selamat datang di sektor "Others". Halaman ini adalah representasi dari minat, bakat, dan fokus pembelajaran saya di dunia teknologi. Alih-alih fiksi ilmiah, di sini saya membagikan teknologi nyata yang saya pelajari dan terapkan sehari-hari: dari baris kode Front-End hingga perakitan sistem Internet of Things.

Setiap blok materi di bawah ini mewakili satu keahlian teknis. Tekan atau klik pada kotak materi yang menarik minat Anda, dan sistem akan langsung mengarahkan Anda ke sumber literatur (dokumentasi resmi/tutorial) tempat saya biasa mendalami ilmu tersebut.

MODUL 01: HTML5 & SEMANTIK WEB

Segala sesuatu di internet dimulai dari kerangka dasar ini. HTML5 bukan sekadar menyusun teks dan gambar, melainkan tentang struktur data yang memiliki makna (semantic web). Penggunaan tag seperti <header>, <main>, dan <section> yang tepat sangat krusial bagi saya untuk memastikan website dapat dibaca dengan baik oleh mesin pencari (SEO) dan teknologi aksesibilitas.

Sebagai langkah pertama dalam Front-End Development, penguasaan HTML memungkinkan saya untuk membangun fondasi arsitektur digital yang kuat sebelum mulai memberikan gaya visual. Semua antarmuka kompleks yang terlihat canggih di portofolio ini, pada dasarnya bersandar pada tulang punggung HTML yang solid.

BACA DOKUMENTASI MDN

MODUL 02: CSS3 & ANIMASI MODERN

Jika HTML adalah tulang, maka CSS adalah kulit dan ototnya. Saya sangat menyukai eksplorasi menggunakan CSS3 untuk menciptakan efek visual yang imersif—seperti desain *dark mode*, efek *glassmorphism* (kaca transparan), hingga pendaran neon *cyberpunk* yang Anda lihat di website ini.

Lebih dari sekadar warna, saya mendalami animasi menggunakan @keyframes dan sistem layout modern seperti Flexbox dan CSS Grid. Memanipulasi *styling* secara manual (Vanilla CSS) memberikan saya kontrol penuh terhadap performa dan detail estetika setiap elemen tanpa harus bergantung pada *framework* eksternal secara berlebihan.

BACA DOKUMENTASI MDN

MODUL 03: JAVASCRIPT LOGIC

JavaScript adalah nyawa dari sebuah website modern. Bahasa pemrograman ini mengubah halaman web statis menjadi aplikasi interaktif yang merespons tindakan pengguna. Saya menggunakan JS untuk mengontrol logika logika web, seperti memvalidasi input formulir Anda, mengatur transisi antar halaman, atau membuat efek *scroll reveal* yang muncul perlahan.

Pemahaman saya tentang Manipulasi DOM (Document Object Model) dan *Event Listeners* adalah kunci utama di sini. JavaScript tidak hanya melatih saya menjadi desainer, tetapi melatih saya berpikir analitis layaknya seorang *Software Engineer* dalam memecahkan masalah.

PELAJARI JAVASCRIPT.INFO

MODUL 04: RESPONSIVE WEB DESIGN

Di era di mana lebih banyak orang mengakses internet lewat ponsel dibanding komputer, desain yang responsif bukan lagi fitur tambahan, melainkan sebuah kewajiban. Saya selalu menerapkan pendekatan *Mobile-First*, yang berarti desain dirancang dan dioptimalkan untuk layar kecil terlebih dahulu sebelum diperbesar ke ukuran desktop.

Dengan menggunakan *Media Queries*, antarmuka yang saya buat akan secara cerdas beradaptasi: merapikan susunan teks, mengubah ukuran font, dan menyembunyikan elemen dekoratif yang terlalu memakan ruang saat dibuka di tablet atau *smartphone*, sehingga kenyamanan pengguna tetap terjaga.

PELAJARI KONSEP RESPONSIVE

MODUL 05: USER INTERFACE (UI) DESIGN

Keahlian teknis *coding* harus diimbangi dengan cita rasa visual. Desain Antarmuka Pengguna (UI) adalah proses di mana saya memilih palet warna, tipografi, dan hierarki visual. Saya aktif mengikuti referensi dan membagikan eksplorasi desain di platform seperti Dribbble maupun TikTok (farras dribble).

Fokus saya dalam UI adalah menciptakan antarmuka yang tidak hanya indah dipandang, tetapi juga menyampaikan identitas *brand* yang kuat. Bagi saya, layar digital adalah kanvas kosong, dan tombol serta menu adalah elemen seni yang harus ditempatkan dengan penuh perhitungan.

EKSPLORASI INSPIRASI DRIBBBLE

MODUL 06: USER EXPERIENCE (UX)

Jika UI adalah tampilan mobilnya, maka UX adalah bagaimana rasanya mengemudikan mobil tersebut. Desain Pengalaman Pengguna (UX) menuntut saya untuk berempati dan berpikir dari sudut pandang target audiens. Apakah sebuah tombol mudah dijangkau oleh jempol? Apakah alur mengisi formulir membingungkan?

Keterampilan UX membuat saya selalu memikirkan alur logika (User Journey). Dari wireframing awal hingga pengujian klik, tujuan utama UX yang saya terapkan adalah menghilangkan *friction* (gesekan/kesulitan) agar pengunjung website bisa mendapatkan informasi atau layanan dengan cara yang paling mudah.

BACA ARTIKEL UX DESIGN

MODUL 07: FIGMA & PROTOTYPING

Sebelum menulis sebaris kode pun, ide-ide digital saya terlebih dahulu diwujudkan melalui platform desain kolaboratif bernama Figma. Alat ini adalah *software* utama saya untuk mendesain *mockup*, membuat ikon vektor, dan merancang layout website secara presisi.

Fitur Prototyping di Figma memungkinkan saya membuat presentasi interaktif di mana klien bisa mengklik tombol dan melihat transisi halaman layaknya aplikasi sungguhan, meskipun sistem belakangnya belum diprogram. Ini sangat mempercepat alur kerja dan revisi proyek.

PELAJARI ALAT FIGMA

MODUL 08: ARDUINO & MIKROKONTROLER

Bakat saya tidak hanya terbatas di dalam layar monitor. Pemrograman Mikrokontroler menggunakan *board* Arduino (seperti Arduino UNO) adalah hobi sekaligus keahlian teknis yang sangat saya gemari. Menggunakan bahasa berbasis C++, saya menulis instruksi untuk otak elektronik berukuran kecil ini.

Ini adalah langkah pertama di mana *coding* saya bisa menyentuh dunia fisik. Dengan Arduino, barisan kode yang saya buat mampu menyalakan lampu, menggerakkan motor, dan membaca suhu ruangan. Ini melatih logika perangkat keras (hardware) saya menjadi sangat tajam.

KUNJUNGI WEB RESMI ARDUINO

MODUL 09: ESP32 / NODE MCU (IoT CORE)

Jika Arduino adalah otaknya, maka modul seperti ESP8266 dan ESP32 adalah sayapnya. Modul mikrokontroler canggih ini telah dilengkapi dengan modul Wi-Fi dan Bluetooth *built-in*, yang menjadikannya mesin utama dalam proyek-proyek Internet of Things (IoT) yang saya buat.

Kemampuannya untuk terhubung ke router internet lokal memungkinkan sistem elektronik saya mengirim data sensor langsung ke server atau menerima perintah nirkabel (wireless) dari *smartphone*. ESP32 adalah jembatan utama penghubung antara alat elektronik dan antarmuka web.

BACA TUTORIAL ESP32

MODUL 10: SENSOR & AKTUATOR ELEKTRONIKA

Dalam merakit alat pintar, pengenalan terhadap komponen keras sama pentingnya dengan menulis kode program. Saya terbiasa mengoperasikan dan mengkalibrasi berbagai jenis sensor: mulai dari Sensor Ultrasonik (untuk membaca jarak), LDR (sensor cahaya), hingga DHT11 (sensor suhu & kelembapan).

Setelah data lingkungan ditangkap oleh sensor, saya memprogram *Aktuator* (seperti Relay, Motor Servo, dan Buzzer) untuk memberikan respons otomatis. Ini adalah fondasi dari otomasi; membuat perangkat mandiri yang bisa mengambil tindakan berdasar kondisi sekitarnya.

EKSPLORASI KOMPONEN ADAFRUIT

MODUL 11: SMART HOME SYSTEM

Salah satu implementasi IoT favorit saya adalah perancangan sistem Rumah Pintar (Smart Home). Menggabungkan keahlian Mikrokontroler dan Relay Listrik, saya merancang sistem yang mampu menghidupkan dan mematikan peralatan rumah (seperti lampu dan kipas angin) dari jarak jauh.

Konsep ini sangat relevan dengan kebutuhan modern akan efisiensi energi dan kepraktisan. Mulai dari sistem saklar berbasis koneksi Wi-Fi internal hingga pemantauan daya listrik berbasis *cloud*, Smart Home adalah miniatur dari visi kota pintar (Smart City).

LIHAT SISTEM HOME ASSISTANT

MODUL 12: SMART GARDEN / PERTANIAN IoT

Tinggal dan bersekolah di kawasan Magelang yang kaya akan agrikultur menginspirasi saya untuk menerapkan teknologi pada bidang pertanian. *Smart Garden* adalah penerapan IoT di mana sensor kelembapan tanah diletakkan di pot tanaman.

Ketika tanah terdeteksi kering, sistem mikrokontroler akan secara otomatis memicu pompa air mini untuk menyiram tanaman hingga batas kelembapan optimal tercapai. Inovasi ini menyatukan dunia hijau dengan dunia digital untuk menghasilkan perawatan tanaman yang presisi dan otomatis.

CARI REFERENSI PROYEK IOT

MODUL 13: INTEGRASI WEB & HARDWARE

Inilah titik puncak di mana semua keahlian saya bertemu. *Web of Things* adalah proses mengintegrasikan data fisik (suhu, jarak, kelembapan) yang ditangkap oleh Arduino/ESP32, dan menampilkannya secara langsung (*real-time*) di antarmuka website buatan saya sendiri.

Saya mampu membuat sebuah dasbor web (Front-End) dengan grafik memukau, yang bertindak sebagai *remote control* pusat untuk mengendalikan aktuator di lapangan. Menggabungkan ranah *Software* dan *Hardware* dalam satu alur ekosistem adalah pencapaian terbesar bagi seorang kreator teknologi.

PELAJARI WEB OF THINGS

MODUL 14: CYBERPUNK & DARK MODE UI

Identitas desain atau *signature style* sangat penting bagi seorang kreator. Saat mendesain web, saya secara khusus memfokuskan diri pada gaya *Cyberpunk* dan arsitektur *Dark Mode* modern. Estetika ini mengandalkan latar belakang gelap kontras dengan teks berwarna *neon* (seperti Cyan dan Emerald).

Lebih dari sekadar terlihat keren atau *hacker-like*, implementasi *Dark Mode* pada website dan aplikasi secara terbukti dapat mengurangi kelelahan mata pengguna saat digunakan di malam hari serta menghemat konsumsi daya baterai pada perangkat berlayar OLED.

LIHAT DESAIN WEB DARK MODE

MODUL 15: GIT & KOLABORASI GITHUB

Dalam dunia pengembangan perangkat lunak profesional, mengamankan baris kode adalah hal krusial. Saya memahami dasar-dasar Git sebagai *Version Control System*, yang memungkinkan saya untuk menyimpan setiap perubahan (*commit*) sejarah dari *codingan* website saya.

Dengan menggunakan platform GitHub, saya bisa mencadangkan proyek saya di *cloud*, membagikan baris kode saya untuk di- *review* secara terbuka, dan berkolaborasi bersama programmer lain tanpa takut merusak kode sumber utamanya. Ini adalah standar industri yang saya pelajari sejak masa sekolah.
